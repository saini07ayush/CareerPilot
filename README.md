# 🚀 CareerPilot – Personalized Interview Prep Module

This branch of **CareerPilot** focuses on generating **AI-powered personalized interview questions** and providing **real-time feedback** on user responses.

## ✨ Features

* 🎯 Generate **role-specific & behavioral questions**
* 📄 Input options:

  * Select a **job role**
  * Upload **resume**
  * Enter **job description**
  * (Optional) choose a **target company**
* 💬 **Interactive interview flow** with follow-up questions
* 📊 **AI feedback & scoring** with improvement suggestions

## ⚙️ How It Works

1. User selects role / uploads resume / enters job description
2. System generates personalized questions
3. User answers questions
4. AI provides **corrections, follow-ups, and score**

## 🛠️ Tech Stack

* Frontend: React / Next.js
* Backend: Node.js / Firebase
* AI: LLM-based evaluation

## 📌 Note

This branch specifically handles **question generation, interaction flow, and answer evaluation**.
