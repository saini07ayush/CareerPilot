# CareerPilot

## Overview

CareerPilot is an AI-powered mock interview platform that helps students and job seekers prepare for interviews through realistic practice and personalized feedback.

---

## Problem

Many candidates struggle in interviews due to lack of confidence and practice, not lack of knowledge. CareerPilot addresses this gap by simulating real interview scenarios.

---

## Features

* AI-based mock interviews
* Resume-based question generation
* Follow-up questions for realistic practice
* Performance feedback and improvement suggestions

---

## How It Works

1. User uploads resume or selects a role
2. AI generates relevant interview questions
3. User answers and receives feedback
4. System helps improve performance over time
