# Resume Extraction & Interview Question Generator

## Overview

This feature allows users to upload their resume and automatically generates interview questions based on its content.
It helps in preparing for mock interviews by focusing on the user’s own skills and projects.

---

## Features

* Upload resume (PDF/Text)
* Extract important details like:

  * Skills
  * Projects
  * Education
  * Experience
* Generate relevant interview questions
* Simple and fast processing

---

## How it Works

1. User uploads their resume
2. The resume is parsed to extract key information
3. Important sections like skills and projects are identified
4. Interview questions are generated based on the extracted data

---

## Example

If a resume contains a project using React, the system may generate questions like:

* How does React handle state management?
* What challenges did you face in your project?
* How did you structure your components?

---

## Tech Stack

* React (Vite)
* JavaScript
* Groq API

---

## Use Case

* Mock interview preparation
* Practice based on your own resume
* Better understanding of your projects and skills
